Release Workflow

Automated releases run on pushes to main.

Steps

  1. Checkout: Fetch repository history.
  2. Python: Configure Python 3.10.
  3. Dependencies: Upgrade pip, install requirements, add wheel.
  4. Git: Set release bot identity.
  5. Semantic release: Version, tag, changelog, release.
  6. Changelog: Run enhancement script.

Reference: Harper Release Bot and the repository.
